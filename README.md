# Interactive Portfolio

An immersive and interactive web-based portfolio showcasing my professional experience, projects, and skills with engaging UI/UX elements.

## Features

- **Interactive 3D Elements**: Physics-enabled 3D sphere with interactive social links and a face that reacts to user input
- **Terminal Interface**: Technical mode with command-line interface for developers
- **Visual Interface**: Non-technical mode with smooth animations and interactive components
- **AI Chat Assistant**: Portfolio assistant to answer questions about skills, experiences, and projects
- **Work Experience City**: Visual representation of work experience as interactive buildings
- **Skill Showcase**: Animated skill icons with customizable animation speeds and styles
- **Project TV**: Interactive display for viewing project information and technologies
- **Mini-games**: Built-in games like DinoRun and Shuffle for an engaging experience
- **Achievements Wall**: Interactive display with animated curtain reveal effect
- **Theme Toggle**: Support for light and dark modes
- **Animation Controls**: Adjustable animation levels for performance optimization

## Technology Stack

- React with TypeScript
- Framer Motion for animations
- Three.js and React Three Fiber for 3D elements
- React Three Cannon for physics simulations
- Redux for state management
- Tailwind CSS and shadcn/ui for styling

## Getting Started

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the portfolio.

## Accessibility

The portfolio is designed with accessibility in mind, including:
- Keyboard navigation support
- Screen reader announcements
- Proper ARIA attributes
- Responsive design for all devices
